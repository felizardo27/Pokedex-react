## Descrição:
Projeto Pokedex desenvolvido em React, usando a api [PokeApi](https://pokeapi.co/), onde traz o id, nome e gif do pokemon
</br>
Possui dois botões para fazer a troca dos pokemons, e uma pesquisa por nome ou por id
